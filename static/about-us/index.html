About Us What's this website for?

This website is intended for anonymously notified people you've been in contact with if you suspect that you have COVID-19 (Coronavirus disease).


Why was this website made?

This website was coded within 3 days for the COVID-19 Global Hackathon 1.0. Feel free to check out the code on Github.
